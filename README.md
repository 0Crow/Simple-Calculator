# Calculator

A small browser-based calculator demo for basic arithmetic operations.

## Tech

- HTML, CSS, JavaScript

## Features

- Addition, subtraction, multiplication, division
- Clear, backspace, and decimal support
- Keyboard support for numbers/operators and Enter for `=`
- Responsive layout for small screens

## Run

- Open `calcu.html` in your browser.

## How it Works

- Click the turn on button.
- Click the number and operators to create expression.
- Click equal button to see the answer.

## Usage examples

- Click `1`, `+`, `2`, `=` to get `3`.
- Enter `12.5 * 4` to get `50`.

## Files

- `calcu.html` — UI markup
- `calcu.css` — styles
- `calcu.js` — calculator logic and keyboard handling

---

**Notes:** Designed for local use. No server required.
